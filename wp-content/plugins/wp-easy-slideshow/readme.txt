Use WP Easy Slideshow plugin to add image slider in WordPress.

== Description ==

User can add slider images through backend and can select list of pages where that particular image should show also create default set.

== Installation ==

1. Upload the `wp-easy-slideshow` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You are done!
4. Go to Dhasboard -> Easy Slideshow to add image.
